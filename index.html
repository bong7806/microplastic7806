미세플라스틱 섭취량 분석 웹앱

이 저장소는 Streamlit 기반의 간단한 데이터 분석 웹앱을 포함합니다.

파일

  • app.py - Streamlit 앱 소스코드
  • requirements.txt - 필요한 파이썬 패키지
  • 섭취량 웹앱 데이터(업로드).csv - 분석할 데이터 (CSV)

실행 방법

git clone <this-repo-url>
cd <this-repo-folder>
pip install -r requirements.txt
streamlit run app.py

앱을 GitHub에 올린 후 GitHub Pages나 Streamlit Cloud에 배포할 수 있습니다.
